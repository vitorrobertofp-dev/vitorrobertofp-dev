## 🚀 Projeto em Destaque

### **Sistema de Reserva de Salas — SENAC GDF**

Uma aplicação web completa para gerenciar a reserva de salas nas unidades do Senac no Distrito Federal.

**Funcionalidades principais:**
- Cadastro de salas, turmas, reservas recorrentes e chaves
- Mapa de ocupação em tempo real (com status: livre / ocupada / em breve)
- Diferentes perfis de usuário: **Administrador**, **Coordenador**, **Instrutor** e **Recepção**
- Controle de solicitações, notificações e histórico de chaves
- Detecção automática de conflitos de reserva
- Tema claro/escuro, pesquisa avançada e filtros em todas as telas
- Totalmente funcional com `localStorage` (sem backend)

**Tecnologias utilizadas:**
- **Front-end:** HTML5, CSS3, JavaScript puro (Vanilla JS)
- **Estilização:** CSS moderno (Flexbox, Grid, variáveis CSS)
- **Dados:** `localStorage` + sistema de Seed completo
- **Outros:** Responsividade, modais, toasts, busca em tempo real

**Prints / Telas do sistema:**
- Login e Landing Page
- Painéis por perfil (Dashboard, Mapa de Salas, Gestão de Turmas, Chaves, etc.)
